
## Installation

```
pkg> add https://github.com/amachino/SuperCowPowers.jl
```

## Usage

### `cowsay`

```
julia> cowsay("Have you mooed today?")
 _______________________
< Have you mooed today? >
 -----------------------
        \   ^__^
         \  (oo)\_______
            (__)\       )\/\
                ||----w |
                ||     ||

```

### `cowthink`

```
julia> cowthink("Have you mooed today?")
 _______________________
( Have you mooed today? )
 -----------------------
        o   ^__^
         o  (oo)\_______
            (__)\       )\/\
                ||----w |
                ||     ||

```
